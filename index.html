What is Messenger E-commerce?

  • ကမ္ဘာ့နောက်ဆုံးပေါ် မာကတ်တင်းနည်းလမ်းဖြစ်ပြီး Messenger မှာပဲ ပစ္စည်းရွေးချယ်တာကနေ Order တင်တဲ့အထိ အစအဆုံး လုပ်ဆောင်နိုင်မယ့် Ecommerce Store (ခေါ်) ဈေးဆိုင်ကြီးတစ်ခု တည်ဆောက်ခြင်း ဖြစ်ပါတယ်။
  • ကိုယ်ရောင်းချမယ့် Products တွေကို သူ့သက်ဆိုင်ရာ Categories အလိုက် စီစီရီရီ ခွဲထားနိုင်မယ်။ ပစ္စည်းတစ်ခုချင်းကို အရောင်/ Size/ Shape တွေအလိုက် ခွဲခြားပြီး ပြသနိုင်မယ်။ ပစ္စည်းရွေးချယ်တဲ့ အဆင့်ကနေ နောက်ဆုံးငွေပေးချေတဲ့အထိ အစမှအဆုံး လုပ်ဆောင်နိုင်တယ်။
  • လျော့ဈေးပေးဖို့အတွက် Coupon Code တွေထုတ်ပေးလို့ ရတယ်။ ဒီအချက်တွေအပြင် အခြားသော စွမ်းဆောင်ရည်တွေ အများကြီးကို လုပ်ဆောင်နိုင်မှာ ဖြစ်ပါတယ်။
